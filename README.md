# MTBApp

Webapp per tracciare uscite in mountain bike, visualizzare velocità, percorso su mappa, altitudine, import/export e navigazione.

## Come usare

1. Apri `index.html` in un browser moderno su dispositivo mobile.
2. Consenti l'accesso alla posizione GPS.
3. Premi `Avvia` per iniziare il tracciamento.
4. Premi `Ferma` per terminare e poi `Esporta .gtx` per salvare la traccia.
5. Usa `Importa .gtx` per caricare una traccia esistente.
6. Cerca un indirizzo, quindi `Naviga` per calcolare una rotta.

## Note

- La mappa usa OpenStreetMap in stile monocromatico.
- Il file esportato è `.gpx`.
- La navigazione utilizza OSRM e la geocodifica Nominatim.
